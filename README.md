## Summary
 This application is used for building an API for a social media network that can handle large amounts of unstructured data utilizing a NoSQL database. The user has the ability to add, delete, get, and update users, friends of users, thoughts, and reactions on those thoughts.  ## Installation
 To install the application, clone the repository and navigate to the project folder on the command line.

 Run npm install from your CLI in order to install Node.js.
